# BlazorSplitViewSample
- Blazorのサーバサイドサンプルをベースにsplit.jsを動作させているサンプルアプリです。

# 動作環境
- Visual Studio 2019
- Blazorサーバテンプレート
- SDK
  - Dotnet core SDK 3.1.300
- ブラウザ
  - Google Chrome
  - Microsoft Edge
  - Internet Explorer 11
